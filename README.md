# Git Basics Lab

Git basics lab exercises for ParoCyber DevSecOps training.

**Author:** Emmanuel Selasie Aggor

## What I practiced
- Initializing a repository
- Staging and committing changes
- Viewing commit history (`log`, `--oneline`, `--graph`, `--stat`)
- Inspecting a specific commit with `git show`
- Diffing across history with `git diff`
- Connecting a local repo to GitHub and pushing
